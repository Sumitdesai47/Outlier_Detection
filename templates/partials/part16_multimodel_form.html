Required Excel format

Timestamp Your_Tag_1 Your_Tag_2 …
2026-01-01 00:00:00 12.4 88.1 numeric
2026-01-01 01:00:00 12.7 87.9 numeric
  • File type: .xlsx only
  • First column: Timestamp
  • Other columns: numbers only (one tag per column)
  • One row per time point; avoid gaps
  • Long files are OK — processing may take a few minutes
